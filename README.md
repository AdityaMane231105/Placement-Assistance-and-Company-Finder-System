# 🎓 Placement Assistance & Company Finder System

## 📌 Project Overview

The **Placement Assistance & Company Finder System** is a full-stack web application designed to help students explore companies, track job applications, and improve placement preparation.

This platform provides real-time company data, skill-based recommendations, and a personalized dashboard to enhance the placement process.


## 🚀 Features

### 🔍 Company Finder

* Search companies by name, role, or skills
* Filter by location, package, and eligibility
* View detailed company information

### 📊 Dashboard

* Track applied companies
* View bookmarked companies
* Monitor application status

### 🧠 Smart Skill Matching

* Match percentage based on user skills
* Helps students identify suitable companies

### 📅 Placement Timeline

* Track application deadlines and interview stages

### 🔐 User Authentication (Planned / Optional)

* Signup/Login system
* Profile management

### 🤖 AI Chatbot (Optional)

* Placement guidance
* Resume and interview tips

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/placement-finder.git
cd placement-finder
```


### 2️⃣ Install dependencies

```bash
cd backend
npm install
```


### 3️⃣ Create `.env` file

Inside `backend/` folder:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```


### 4️⃣ Run backend server

```bash
node server.js
```


### 5️⃣ Run frontend

* Open `index.html`
* OR use Live Server in VS Code
